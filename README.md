# :construction_worker: How work ?
```bash
# Clone this repository
$ git clone https://github.com/KaikySantos/3d-solar-system.git

# Access the project folder
$ cd 3d-solar-system

# Install the dependencies
$ yarn install

# Run the application
$ yarn start
```
Go to http://localhost:3000/ to see the result.
